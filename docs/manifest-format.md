# LVK Update Feed manifest format

Этот репозиторий хранит только публичные JSON-файлы с версиями приложений.

Код апдатера хранится отдельно в приватном репозитории `LVK-Updater`.

## apps.json

Общий список приложений и ссылок на их manifest-файлы.

## manifests/findfile.json

Manifest конкретного приложения FindFile.

Основные поля:

- `appId` — внутренний идентификатор приложения.
- `latestVersion` — последняя опубликованная версия.
- `channel` — канал обновления: `stable`, `beta`, `dev`.
- `mandatory` — обязательное обновление или нет.
- `mainExe` — главный exe-файл приложения.
- `package.url` — ссылка на zip-архив релиза.
- `package.sha256` — контрольная сумма архива.
- `package.size` — размер архива в байтах.
- `notes` — список изменений.
