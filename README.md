# LVK-Update-Feed

Публичный канал обновлений для приложений LVK.

Здесь хранятся только JSON manifest-файлы:

```text
apps.json
manifests/findfile.json
```

Приложения читают эти файлы, чтобы узнать актуальную версию и ссылку на архив обновления.

Важно: установленные приложения не пишут в этот репозиторий. Manifest-файлы обновляются автоматически через GitHub Actions из репозиториев приложений, например из `FindFile_MVP`.
